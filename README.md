# منصة رِواق أكاديمي

واجهة موقع أكاديمي ثابتة مبنية بـ HTML وCSS وJavaScript، قابلة للنشر على GitHub Pages.

## الملفات

- `index.html` الصفحة الرئيسية
- `styles.css` ملف التصميم
- `script.js` التفاعل والبيانات التجريبية

## النشر على GitHub Pages

1. أنشئ Repository جديداً على GitHub.
2. ارفع الملفات الثلاثة إلى المستودع.
3. افتح Settings ثم Pages.
4. اختر Branch: `main` ثم `/root`.
5. اضغط Save.
6. سيظهر رابط الموقع بعد دقائق.

## فلسفة المنصة

المنصة تقدم خدمات دعم وإرشاد ومراجعة وتحرير وتنسيق وتدريب أكاديمي، وليست منصة لبيع الرسائل أو الأبحاث الجاهزة.
